# 随机变量组合分析程序 (C语言版本)

## 程序功能

本程序实现了以下功能：
1. 生成服从贝努利二项分布的随机变量(-1,1)
2. 生成标准正态分布N(0,1)的随机变量
3. 将两个随机变量相加
4. 统计组合变量的概率分布
5. 输出直方图和统计信息

## 编译和运行

### 方法1：使用Makefile
```bash
# 编译程序
make

# 运行程序
make run

# 清理编译文件
make clean
```

### 方法2：直接使用gcc
```bash
# 编译
gcc -Wall -Wextra -std=c99 -O2 -o random_variable_analysis random_variable_analysis.c -lm

# 运行
./random_variable_analysis
```

## 程序输出

程序会输出：
1. 贝努利变量的统计信息
2. 正态变量的统计信息
3. 组合变量的详细统计信息（均值、标准差、方差、最值）
4. 理论分析说明
5. 直方图（文本形式）
6. 理论值与实际值的对比

## 理论背景

如果 X ~ Bernoulli(-1,1) 且 Y ~ N(0,1)，那么 Z = X + Y 的分布为：
- E[Z] = 0
- Var[Z] = 2
- Z 的密度函数为：f(z) = 0.5 × φ(z+1) + 0.5 × φ(z-1)

其中 φ 是标准正态分布的密度函数。

## 技术实现

- 使用标准C库的rand()函数生成均匀分布随机数
- 使用Box-Muller变换生成标准正态分布
- 手动实现统计计算（均值、方差、标准差）
- 创建文本形式的直方图可视化

## 系统要求

- C99兼容的编译器（如gcc、clang）
- 数学库支持（-lm标志）
- 支持中文输出的终端
